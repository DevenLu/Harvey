# Harvey 
Harvey is a stubbing library written completely in Swift. Main goals:
- [ ] Stubbing based on given request properties (url/method etc.)
- [ ] Removing stubs
- [ ] Compatible with SPM
- [ ] Record/save/use recorded stubs
- [ ] Moya plugin to easily handle Harvey

# Contributions
